# Vue Storefront 2 integration with ecoshop

This project is a ecoshop integration for Vue Storefront 2.

This integration is currently a **Beta** and not ready for production usage. You can help us to make the integration production-ready faster by reporting bugs and contributing to the code at the [repository issues page](https://github.com/vuestorefront/ecoshop/issues).
